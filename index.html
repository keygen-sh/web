Use your favorite languages

Easily use the Keygen API in the programming language of your choice. Download and install libraries for Node.js, Go, Ruby, PHP, and more.

const keygen = require("keygen")("1f2b43a8-b8a2-4b90-b248-9184368cda57")

// Login and create a session
const session = await keygen.sessions.create({
  email, password
})

// Create a license policy
const policy = await session.policies.create({
  product: keygen.products.get({ name: "Awesome Product" }),
  duration: time.ms("6 months"),
  name: "Product Add-On"
})

// Create your first user
const user = await session.users.create({
  name: "John Doe",
  email: "john.doe@example.com",
  password: "secret"
})

// Create a license for the new user
const license = await user.licenses.create({
  policy
})

// Validate their new license
const { isValid } = await license.validate()

if (isValid) {
  // Do something
} else {
  // Do something else
}

FAQs

  • What is Keygen?

    Keygen is a hosted software-as-a-service (SaaS) licensing API, meaning it will require an active internet connection to validate licenses. Keygen was designed to make licensing easy for desktop and web-based products. Keygen allows you to build and release your product quickly by offering a simple licensing API that is easy to implement.

  • Why was Keygen created?

    With the rise of web apps and JavaScript-based desktop apps built on Electron and NW.js, a void opened up for an easy licensing solution like the ones provided by Apple and Google for their mobile platforms. Keygen was created to fill that void.

  • Will you offer on-premise API licensing?

    Yes. We're very interested in talking with companies that would rather host the Keygen licensing API in-house. Get in touch.

  • Is data within Keygen secure?

    Yes. We strive to ensure that the data stored within Keygen is as secure as possible. We never store passwords as plain text – they are always hashed and salted securely using bcrypt. We also do the same with all access tokens, and offer the ability to hash license keys as well. All network traffic is encrypted over TLS with at least 128-bit AES encryption.

  • Is Keygen pen tested?

    Currently, no. But we have a third party security testing of Keygen scheduled to be performed by an independent and reputable security consulting firm before we leave beta. We take security very seriously, and would love to answer additional questions, if you happen to have any. Get in touch.

  • Does Keygen prevent sharing licenses?

    Keygen offers endpoints to track machine usage, and policies can be set to strict, which will ensure your users do not go over their machine limit. A license can be set to floating, which will allow the license to be valid across multiple machines.

  • Does Keygen prevent cracking?

    No. Keygen does nothing to prevent a user from removing your licensing logic altogether if they have access to your application's code. In the end, you will have to make the choice on how to deal with cracking by doing things such as: requiring a valid license for updates and support, etc.

  • When will Keygen be released?

    We are in the process of writing our documentation, getting started guides, and our client libraries. As soon as that is finished, we will launch a short beta before finally releasing Keygen out into the wild.

  • Does Keygen handle payments?

    Since products handle payments in a variety of ways, we have created a webhook system so that you can easily integrate your payment system with our API. Implementation is similar to how you could implement Stripe's webhook system.

  • Will Keygen offer open source plans?

    Most definitely. If you want to use Keygen for an open source project, send us a link to your project and we will review it to determine if it's eligible for a free open source plan.

  • Can I use Keygen for an iOS app?

    Unfortunately, it isn't possible at this time. Apple’s developer terms require that purchases related to an app, such as premium content and add-ons are managed via their native In-App Purchase API.

  • Can I use Keygen for an Android app?

    Unfortunately, it isn't possible at this time. Google's developer policy requires that purchases related to an app, such as premium features and add-ons are managed via their native In-App Billing API.
