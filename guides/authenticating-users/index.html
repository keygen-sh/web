Authenticating Users

Use Keygen's API and your client-side code to authenticate users in your product. If you need help after reading this, can reach out to us anytime at support@keygen.sh.

This tutorial is written in JavaScript, but you can follow along in any other language that you're comfortable in. Throughout this tutorial, you will see placeholders such as {ACCOUNT} within the code examples that will need to be replaced with your account's slug or ID.

const credentials = btoa("{EMAIL}:{PASSWORD}")
const response = await fetch("https://api.keygen.sh/v1/accounts/{ACCOUNT}/tokens", {
  method: "POST",
  headers: {
    "Content-Type": "application/vnd.api+json",
    "Accept": "application/vnd.api+json",
    "Authorization": `Basic ${credentials}`
  }
})

const { data, errors } = await response.json()
